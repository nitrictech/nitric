# .NET - Topic()

Creates a new Topic.

```csharp
using Nitric.Sdk;

var updates = Nitric.Topic("updates");

Nitric.Run();
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>
