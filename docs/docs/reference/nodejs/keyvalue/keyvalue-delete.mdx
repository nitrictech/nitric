# Node.js - kv.delete()

<Note>
  This is reference documentation for the Nitric Node.js SDK. To learn about
  Key/Value Stores in Nitric start with the [Key/Value Stores docs](/keyvalue).
</Note>

Delete key from a key value store.

```javascript
import { kv } from '@nitric/sdk'

const profiles = kv('profiles').allow('delete')

await profiles.delete('profile-1a2b3c')
```

## Parameters

<Properties>
  <Property name="key" required type="string">
    The key that references the key value pair that should be deleted.
  </Property>
</Properties>

## Examples

### Delete a key

```javascript
import { kv } from '@nitric/sdk'

const profiles = kv('profiles').allow('delete')

await profiles.delete('profile-1a2b3c')
```
