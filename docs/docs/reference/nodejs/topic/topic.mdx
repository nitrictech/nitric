# Node.js - topic()

<Note>
  This is reference documentation for the Nitric Node.js SDK. To learn about
  Topics in Nitric start with the [Async Messaging docs](/messaging#topics).
</Note>

Creates a new Topic.

```javascript
import { topic } from '@nitric/sdk'

const updates = topic('updates')
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>

## Examples

### Create a new topic

```javascript
import { topic } from '@nitric/sdk'

const updates = topic('updates')
```
