# Node.js - topic()

Creates a new Topic.

```javascript
import { topic } from '@nitric/sdk'

const updates = topic('updates')
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>

## Examples

### Create a new topic
