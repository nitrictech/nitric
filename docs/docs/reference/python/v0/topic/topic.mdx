# Python - topic()

Creates a new Topic.

```python
from nitric.resources import topic
from nitric.application import Nitric

updates = topic('updates')

Nitric.run()
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>
