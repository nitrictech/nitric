# Python - job()

<Note>
  This is reference documentation for the Nitric Python SDK. To learn about
  Batch Services in Nitric start with the [Batch docs](/batch).
</Note>

Creates a new Batch Job.

```python
from nitric.resources import job

analyze = job("analyze")
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The unique name of this Batch Job within the app. Subsequent calls to `job`
    with the same name will return the same object.
  </Property>
</Properties>

## Examples

### Create a Job

```python
from nitric.resources import job

analyze = job("analyze")
```

### Create a Job with permissions to submit jobs

```python
from nitric.resources import job

analyze = job("analyze").allow("submit")
```
