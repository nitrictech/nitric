# Go - NewTopic()

<Note>
  This is reference documentation for the Nitric Go SDK. To learn about Topics
  in Nitric start with the [Async Messaging docs](/messaging#topics).
</Note>

Creates a new Topic.

```go
import (
  "github.com/nitrictech/go-sdk/nitric"
)

func main() {
  updates := nitric.NewTopic("updates")

  nitric.Run()
}
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>

## Examples

import (
