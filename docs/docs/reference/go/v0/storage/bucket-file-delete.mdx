# Go - Bucket.File.Delete()

Delete a file from a bucket.

```go
import (
  "context"
  "fmt"

  "github.com/nitrictech/go-sdk/nitric"
)

func main() {
  bucket, err := nitric.NewBucket("bucket-name").With(nitric.BucketDeleting)
  if err != nil {
    return
  }

  err := bucket.File("cat.png").Delete(context.TODO())
  if err != nil {
    return
  }

  nitric.Run()
}
```
